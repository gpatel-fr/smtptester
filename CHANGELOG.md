# Change Log

## 1.1.0: 2022-05-26

- Use QT6 (PySide6)
- Use [Poetry](https://python-poetry.org/)
- Added short options:
  - `-l`: Log level
  - `-s`: Sender
  - `-m`: Message
  - `-d`: DNS host
  - `-h`: SMTP host
  - `-u`: SMTP auth user
  - `-p`: SMTP auth password
- Removed colored logs in GUI
- Added type annotations

## 1.0.1: 2020-03-22

- Fix for pip not installing dependencies

## 1.0.0: 2020-02-29

- Complete rewrite
  - Support Python 3.7+
  - Add unit tests
  - Add CLI interface
  - Switch from wxWidgets to QT5 (PySide2)

## 0.1.6: 2012-08-27

- Fix incorrect name in title bar

## 0.1.5: 2012-08-27

- Fix missing icon on some platforms

## 0.1.4: 2012-08-24

- Add support for macOS

## 0.1.3: 2011-01-28

- Miscellaneous refactoring
- Use proper semantic versions

## [0.]1.2: 2010-04-24
